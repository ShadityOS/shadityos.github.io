What is it?

What is ShadityOS?

ShadityOS is a Debian-based operating system that runs on the Raspberry Pi 4b and 400.

ShadityOS is developed and managed on Github You can follow our development here.

ShadityOS comes jam packed with lots of programs and customization options so any user can find the right look for their desktop.

If you have found a bug or a problem, please feel free to make an issue on our Github. We will attend to it ASAP.

Our Github

Our Github

On our Github page, we manage many different projects that are incorporated into ShadityOS. We spend our time making and improving ShadityOS for the community.

Any help with debugging and adding features is welcome. With the community helping us ShadityOS remains a stable and reliable operating system for everybody to use.

You can visit our Github today at https://Github.com/ShadityOS
